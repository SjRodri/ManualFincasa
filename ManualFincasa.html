Introducción

Antecedentes

El sistema que se presenta ha sido desarrollado para satisfacer una necesidad de automatización dentro del emprendimiento "Cabañas del Campo, FINCASA", ubicado en la localidad de San Pedro Zacapa, Santa Bárbara. Este sistema ha sido diseñado para agilizar los procesos internos, mejorar la calidad de la atención y reducir el tiempo necesario para realizar una reserva y gestionar la facturación.

Durante el desarrollo, nos enfocamos en crear interfaces amigables, intuitivas y fáciles de usar. También nos aseguramos de cubrir todas las necesidades expuestas por los propietarios del complejo de cabañas, con el objetivo principal de mejorar el proceso de reserva para todos sus clientes.

Objetivos del Sistema:

  • Manejo y visualización de la lista de clientes con sus respectivos datos.
  • Gestión del proceso de reserva, control de las fechas y reservas por cada cabaña del complejo.
  • Automatización del proceso de facturación, incluyendo un apartado de cotización.
  • Gestión y manejo de las cabañas y servicios prestados por FINCASA.
  • Un sistema que cumpla con los requisitos establecidos por los clientes dueños del emprendimiento FINCASA.

Introducción:

La razón fundamental de nuestro sistema es mejorar los procesos realizados en "FINCASA" en cuanto al manejo de clientes, fechas para reservas y facturación. Anteriormente, estos procesos no estaban automatizados y se realizaban manualmente con papel y lápiz. Nuestro objetivo principal durante el desarrollo de este sistema fue la correcta automatización de estos procesos, facilitando el trabajo del personal.

¿A quién está dirigido el manual?

Este manual está dirigido a las personas administradoras del emprendimiento "Cabañas del Campo, FINCASA" y demás empleados, ya que son la parte principal y fundamental de nuestro sistema. Les permitirá manejar las reservaciones de su complejo de manera más segura y ágil.

Generalidades del Sistema

Descripción del proyecto

El sistema desarrollado para “Cabañas del Campo, FINCASA”, se compone de distintos módulos, los cuales en su mayoría son dependientes.

En primera instancia tenemos una medida de seguridad, la cual se ha diseñado como un inicio de sesión en el cual el usuario podrá ingresar sus credenciales al sistema para validar que este pertenece al mismo, siendo éste nuestro primer módulo independiente.

Nuestro sistema está realizado para solo aceptar un rol de usuario, el cual podríamos llamar Administrador, ya que este accede a todos los elementos y bondades de nuestra aplicación.

Una vez estemos dentro del sistema nos podemos encontrar con un menú muy atractivo visualmente que nos mostrará todos los procesos que podemos realizar. Existen seis opciones diferentes, dentro de las cuales se encuentran:

  • Realización de reservación
  • Consulta de fechas
  • Facturación y consulta de reservas
  • Cotización
  • Mantenimiento de cabañas
  • Mantenimiento de servicios

Dentro de todas estas opciones está contemplado nuestro sistema, el cual permite un manejo completo de las reservas, permitiendo exportar facturas de las mismas. Dentro de nuestro elemento principal que son las reservaciones, podemos editar las reservas, eliminar elementos como cabañas o servicios, y modificar estos mismos. Aparte de poder enviar la reservación por medio de correo electrónico al cliente mediante un mensaje completamente automatizado.

Otro punto a destacar, pero no menos importante, es el manejo de los servicios y cabañas, los cuales pueden ser modificados, eliminados o, si el usuario lo decide, puede agregar uno nuevo. Estos elementos son importantes ya que nos permiten tener lista nuestra aplicación en caso de que el complejo de cabañas sufra cambios en cuanto a sus productos y servicios.

El alcance de nuestra aplicación va desde el registro de clientes, consulta de fechas, realización de reservas, confirmación de reserva mediante correo electrónico, hasta la generación de reportes como ser facturas y cotizaciones, las cuales serán almacenadas de forma local para su posterior consulta cuando se requiera.

Requerimientos Técnicos del Sistema, Instalación y Configuración

Para la instalación del sistema se requerirá de lo siguiente:

  • Conexión de internet estable obligatoria.
  • La rapidez de descarga o el ancho de banda de la conexión no importan, pero sí influyen en cuanto tarde la descarga en finalizar.
  • Computadora que tenga el sistema operativo Windows instalado.
  • Versión Windows 10 o posterior.
  • 4GB de RAM como mínimo.
  • Computadora de Escritorio o Laptop.
  • Pantalla que contenga una Resolución de 1024x768 o superior.
  • Periféricos: Teclado y ratón u otro dispositivo de entrada.
  • Procesador de al menos 2 Ghz.
  • La memoria de disco duro requerida dependerá de la cantidad de archivos PDF que desea ingresar en su computador.
  • Lugar para poder descargar el programa, la cual se le puede facilitar tanto en la plataforma de Whatsapp como también en la plataforma de Google Drive.

Instalación

Para la primera instalación en la computadora principal, además de los requerimientos antes mencionados, se ocupará un breve periodo de soporte técnico en el que el grupo presente hará las respectivas instalaciones de los recursos necesarios para la ejecución del sistema, como lo serían: el gestor de base de datos SQL Server Management Studio, la creación de una cuenta de servicio, entre otros. Se recurre a decir que estos recursos sólo serán ocupados en la computadora principal y la descarga de los mismos se harán por parte de nuestro grupo como un soporte técnico dado al sistema.

Para la utilización del programa no se ocuparía más que una decente conexión a internet y revisar que la base de datos está en ejecución. Si la base de datos no llegara a estar en ejecución, por favor refiérase al apartado de manejo de errores para solucionarlo.

Pasos para la instalación del programa:

Paso #1: Para el primer paso dentro de la instalación del sistema entraría lo que es descargar el instalador del programa de un sitio como lo podría ser Whatsapp o en un futuro Google Drive.

Paso #2: Una vez se haya descargado la carpeta comprimida, se irá al apartado de descargas dentro de su explorador de archivos en donde podrá ver su descarga.

Paso #3: Extraer los archivos: Haz clic derecho en el archivo comprimido FINCASA y selecciona la opción Extraer todo. En algunos casos, si tienes instalado un programa específico para archivos comprimidos como WinRAR o 7-Zip, podrías tener que seleccionar Abrir con y luego elegir el programa.

Paso #4: Para asegurarse de que la carpeta no se pierda le dará click derecho una vez a la carpeta hasta que quede resaltado en azul como en la imagen anterior. Después, presionará en su teclado las teclas “ctrl” y la tecla “x” a la vez y se irá al apartado de PC.

Paso #5: Le hará doble click en donde dice Windows (C:) (En vez de la letra C puede haber una letra D u otras pero es lo mismo). Después de entrar en la pantalla presione la tecla “ctrl” y la tecla “v” al mismo tiempo para pegar el archivo.

Paso #6: En la pantalla en la que se encuentre la carpeta debería quedar así, si ese no es el caso váyase a la sección de descargas y repita los pasos desde el Paso #4.

Paso #7: Busca el archivo de instalación: Una vez extraídos los archivos, abre la carpeta que se creó a partir de la extracción. Dentro de ella, busca un archivo llamado SETUP.exe o simplemente SETUP.

Paso #8: Sigue el asistente de instalación: Sigue las instrucciones proporcionadas por el asistente de instalación, selecciona la opción instalar.

Paso #9: Una vez se complete la instalación, automáticamente se ejecutará la aplicación.

Paso #10: Cuándo queramos volver a iniciar la aplicación podemos buscarlo directamente desde el buscador de Windows.

Entrada y Salida del Sistema

Entrada del sistema - Inicio de sesión

Al iniciar el sistema, lo primero que podrá ver será nuestro inicio de sesión el cual posee dos cajas de texto que solicitan las credenciales necesarias para poder ingresar a la aplicación. En el primer campo deberá introducir el nombre del usuario que se le fue asignado, en la segunda caja de texto deberá introducir la contraseña respectiva para ese usuario, luego de haber llenado estos campos de forma correcta podrá presionar el botón de “Ingresar” de esta manera estará entrando al sistema, en el caso de que sus credenciales sean correctas se mostrará el menú con todas las opciones que puede realizar.

Salida del sistema

En todos los formularios del sistema se encontrará con un botón ilustrado con una “X” la cual sirve para indicar que si acciona este botón estará cerrando completamente la aplicación.

En algunas otras pantallas o formularios podrá encontrarse con elementos que le permiten salir del formulario o pantalla en la que se encuentra.

Manejo de Errores

Tabla de Errores

1. Si la aplicación parece estar inactiva, tenga paciencia y permítale completar sus tareas actuales. Durante la carga o descarga de archivos, especialmente si son de gran tamaño, es común experimentar cierta lentitud, la cual también puede verse afectada por la velocidad de su conexión a Internet.

2. Si encuentra que no puede utilizar la aplicación mientras maneja archivos, y si la aplicación se detiene o permanece en la misma pantalla por un tiempo prolongado, superior a treinta minutos, debería verificar si su máquina se encuentra detenida. Este tipo de comportamiento suele estar asociado con problemas de rendimiento.

Mensaje de Error Descripción Solución Sugerida
Error de Conexión La aplicación no puede establecer conexión con el servidor. Verificar la conexión a Internet y volver a intentarlo.
Error de Autenticación Nombre de usuario o contraseña incorrectos. Verificar las credenciales e intentar nuevamente.
Error de Reserva No se pudo completar la reserva. Asegurarse de que todos los campos requeridos estén llenos y en el formato adecuado. Volver a intentar la reserva.
Error de Tiempo de Espera La operación excedió el tiempo de espera. Verificar la conexión a Internet y volver a intentar la operación.
Error Interno de Aplicación Ocurrió un error inesperado en la aplicación. Reiniciar la aplicación y, si el error continúa, reinstalar la aplicación o contactar al soporte técnico.

Si la aplicación se congela o no avanza más allá de la pantalla de inicio de sesión, es posible que exista un inconveniente con el servidor SQL. Para solucionar este problema, siga los pasos a continuación:

Paso #1: Ingresar la palabra "service" o "servicios" en el buscador de Windows y seleccionar la primera opción que aparece en la imagen.

Paso #2: Haga clic en el archivo mencionado y desplácese hacia abajo hasta localizar una línea que indique “SQL Server (SQLEXPRESS)”.

Paso #3: Presionar clic derecho y darle la segunda opción que sería "detener".

Paso #4: Presionar clic derecho y darle a la primera opción que sería "iniciar".

Aguarde a que finalice el proceso y cierre todas las aplicaciones o ventanas que haya abierto. Si el problema se origina en el servidor de bases de datos, debería estar resuelto en este punto.

Si se está experimentando problemas en el uso de la aplicación como por ejemplo, un rendimiento lento, conexiones bloqueadas, o comportamientos inesperados, un reinicio puede limpiar el estado actual del servidor y solucionar problemas de funcionamiento sin identificar.

Glosario de Términos

Término Definición
Autenticación Proceso mediante el cual un sistema verifica la identidad de un usuario, generalmente a través de un nombre de usuario y contraseña, para conceder acceso a sus servicios.
Base de Datos Conjunto organizado de datos almacenados digitalmente en una computadora que son accesibles de manera eficiente para su consulta y actualización.
Cabañas del Campo, FINCASA Emprendimiento dedicado al alquiler de cabañas, cuyos procesos de reservación y facturación han sido automatizados mediante un sistema informático.
Cotización Documento que detalla el precio estimado de los servicios o productos ofrecidos, en este caso, el alquiler de cabañas.
Facturación Proceso de creación de un registro oficial (factura) que detalla los productos o servicios vendidos al cliente, junto con los precios y condiciones.
Gestión de Reservas Proceso de organizar y controlar las reservaciones de cabañas, incluyendo la asignación de fechas, seguimiento de disponibilidad y confirmación de reservas.
Interfaz de Usuario Componente de un sistema que permite la interacción entre el usuario y el ordenador, ofreciendo elementos visuales y funcionales para navegar y operar el sistema.
Mantenimiento de Cabañas Proceso de administrar y actualizar la información relativa a las cabañas disponibles, incluyendo la adición, modificación o eliminación de datos en el sistema.
Mantenimiento de Servicios Gestión y actualización de los servicios ofrecidos por el complejo, tales como limpieza, alimentos, entre otros.
PDF (Formato de Documento Portátil) Formato de archivo utilizado para presentar e intercambiar documentos de manera fiable, independientemente del software, hardware o sistema operativo.
Resolución de Pantalla Refiere a la cantidad de píxeles que se pueden mostrar en la pantalla de un dispositivo, medida en términos de ancho y alto (por ejemplo, 1024x768).
SQL Server Management Studio (SSMS) Herramienta integrada de entorno para administrar cualquier infraestructura de SQL, desde bases de datos SQL Server hasta Azure SQL Database.
Sistema Operativo Windows Serie de sistemas operativos gráficos producidos por Microsoft, utilizados en computadoras personales y servidores.
Soporte Técnico Servicio que proporciona asistencia con el hardware o software de una computadora, en este caso, ayudando a la instalación, configuración y resolución de problemas del sistema de gestión para FINCASA.
Tabla de Errores Listado que describe los posibles errores que pueden surgir durante el uso del sistema, proporcionando detalles sobre la causa y las soluciones recomendadas.
